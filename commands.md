# Command Reference

## General
| Command | Description |
| :--- | :--- |
| `/help` | Show list of available commands. |
| `/pricing` | View subscription plans and pricing. |
| `/setup` | Initialize bot settings for the server. |

## Verification
| Command | Description |
| :--- | :--- |
| `/verify` | Start the verification process (Solana). |
| `/verify-evm` | Start the verification process (EVM). |
| `/verify-ton` | Start the verification process (TON). |
| `/unverify` | Unlink your wallet from the server. |
| `/setup-verification-panel` | Create a "Verify" button panel in a channel. |
| `/verification-status` | Check your current verification status. |

## Configuration (Admin)
| Command | Description |
| :--- | :--- |
| `/nft-config` | Add/Remove NFT collection requirements. |
| `/token-config` | Add/Remove Token requirements. |
| `/staking-config` | Configure staking requirements. |
| `/allowlist` | Manage wallet allowlists. |
| `/config` | View current server configuration. |

## Moderation
| Command | Description |
| :--- | :--- |
| `/add-prohibited` | Add a word to the blocklist. |
| `/remove-prohibited` | Remove a word from the blocklist. |
| `/config-link-moderation` | Toggle link filtering. |
| `/add-link-channel` | Whitelist a channel for links. |
| `/config-mentions` | Toggle mention protection. |
| `/exempt-role` | Exempt a role from auto-mod. |

## Social Feeds
| Command | Description |
| :--- | :--- |
| `/feed add` | Add a Twitter or Telegram feed. |
| `/feed remove` | Remove a social feed. |
| `/feed list` | List active feeds. |
